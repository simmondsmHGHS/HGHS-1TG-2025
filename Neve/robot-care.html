Here's how you can ensure you take good care of the robots:

When using the robot, you will find a screen that has six buttons.

The most important buttons are the cancel and enter buttons. The other four are used for moving around (up, down, left, and right).

The cancel button is used to cancel any program currently running and to turn off the robot. If a program is running and you want it to stop, press the cancel button and it should stop.

The enter button is used to choose a program or setting. Pressing it will choose and start the program.

The screen is used to change settings or select a program. When using a robot, you only need to use the first two screens.

The first screen has all of the recent programs in it. Select the program that you want to run by using the up and down buttons, then press the enter key to select it.

You can get to the second screen by using the right arrow. This is where all of the programs are kept in folders. When you select a program there, you need to press the down button and select the program with the play button on the left of the name. This will play the program.

The robot has been set up in a cuboid base with all of the sensors that you will need attached to it already. Each sensor and motor is connected to the robot in specific spots.

These sensors and motors are:

  • Ultrasonic sensor

  • Colour sensor

  • Large motor

Do not change or move any of these sensors and motors. They are set up in specific locations to work properly on robots, and moving them will affect you and the people who use the robot after you.
